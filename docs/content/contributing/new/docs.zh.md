### 为文档做贡献

贡献文档与编写代码同样重要，甚至更重要。我们使用 [Hugo](https://gohugo.io/) 来运行这个网站。如需在本地运行，请按以下步骤操作：

1. 安装 Hugo 二进制文件：https://github.com/gohugoio/hugo#choose-how-to-install
2. `cd ./docs && hugo server`

Hugo 服务器将在 http://localhost:1313 上运行，它会自动监测文件变化并实时更新网站。

或者，您可以按[此处](https://github.com/gomods/athens/blob/main/DEVELOPMENT.md#build-the-docs)所述运行我们的自定义 Docker 镜像
