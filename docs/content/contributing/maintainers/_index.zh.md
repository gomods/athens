有[很多方式可以为 Athens 项目做贡献](./participating)，成为维护者只是其中之一。Athens 项目的维护者每周大约花费五小时进行项目开发和维护。他们的职责包括：

- 帮助组织开发会议（如帮助组织议程）
- 在可能时推广项目并建设社区（如在可能时进行演示、写文章……）<sup>2</sup>
- 分类问题（如添加标签、促进讨论、最终决定）
- 组织和促进 PR 审查（如提示社区成员、贡献者和其他维护者进行审查）
- 帮助为所有项目参与者培养安全和热情的环境。
这包括执行我们的行为准则。我们遵守[贡献者公约](https://www.contributor-covenant.org)，如果您还没有阅读，可以在[此处](https://www.contributor-covenant.org/version/1/4/code-of-conduct)阅读（英文版）。

我们很乐意与您合作建设 Athens，但这是一个需要承诺的项目，像 Athens 这样规模和复杂性的项目所要求的。如果您感兴趣，[在 Slack 上加入我们](https://gophers.slack.com/archives/C9LRAQN8N)。

## 成为维护者的好处

- 免费使用 GitHub CoPilot
- 成就感
- 在安全协作环境中的有趣、复杂的工作
- 与非常聪明的人交流的机会



## 不做维护者也可以贡献

如果您无法维护，我们始终寻找人们来：
- 开发功能
- 修复错误
- 分类问题
- 提供支持

您只需访问我们的 [GitHub 页面](https://github.com/gomods/athens) 即可完成以上所有事情！
