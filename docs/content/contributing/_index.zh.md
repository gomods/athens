欢迎，Gopher！我们很高兴您考虑为 Athens 贡献力量。在您开始之前，我们想先向您介绍我们的社区。

我们社区有一些硬性规则，如[行为准则](https://github.com/gomods/athens/blob/main/CODE_OF_CONDUCT.md)，但我们不会预先制定规则，而是尝试牢记一个共同理念来帮助我们所有人做出决策，并在需要时制定新规则。

## 我们的理念文档

这些文档的[理念部分](/contributing/community/philosophy/)详细阐述了我们。如果您参与我们的项目，我们鼓励您阅读。如果您只是浏览，这里是简要总结：

- **彼此友善** - 人 > 代码，始终如此，每一次都如此。我们认为专注于彼此的关系，最终会得到更好的技术、更好的代码和更好的社区。
- **简化开发和测试** - 尽可能地简化。这有助于我们减少认知负担，以便专注于 Athens，而不是搭建开发/测试环境。
- **关注社区** - "如果你想走得快，独自走。如果你想走得远，一起走。"我们所有人都努力遵循这句谚语。我们尽可能让更多人参与，（几乎）100% 透明地做所有事情，并且我们互相信任去做正确的事。

## 从这里开始

我们希望您喜欢所见内容，很乐意您参与进来。

如果您熟悉 Git 和 GitHub 基础知识，请阅读[如何参与 Athens 社区](./community/participating) 以了解我们如何组织以及如何开始。

如果您是 Git 和 GitHub 的新手，请查看我们的[新手贡献者指南](./new)，然后继续阅读上面提到的如何参与文档。

最后，如果您是项目的新维护者，我们在 [./maintainers](./maintainers) 为您准备了一些文档。
